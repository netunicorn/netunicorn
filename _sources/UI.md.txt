# User Interface

The user interface could be provided by administrator to users for getting the current information about experiments, nodes and compilations. In particular, it shows the following information:
- A list of active (running) experiments, including which nodes are used in each experiment.
- A experiments history for last 14 days.
- A list of nodes that are curerntly locked by the user's experiments.
- A list of user's compilations that are currently running.

For users having `sudo` privileges, the user interface provides this information about all users of the system.

The user interface URL is provided by the administrator or located at the default URL: `http://<system url>:8080/`.
